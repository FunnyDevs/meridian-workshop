# Pricing

**RFP #MC-2026-0417 — Meridian Components Inventory Dashboard Modernization**

---

## Struttura dell'offerta

Proponiamo un contratto **fixed-fee** per i required item (R1–R4), con un not-to-exceed separato per i desired item (D1–D3) soggetto ad approvazione esplicita di Meridian prima dell'avvio.

---

## Required Items (R1–R4)

| Item | Descrizione | Fee |
|------|-------------|-----|
| R4 | Architecture documentation | €4.000 |
| R3 | Automated browser testing (Playwright) | €6.000 |
| R1 | Reports module remediation | €8.000 |
| R2 | Restocking recommendations (design + build) | €18.000 |
| **Totale required** | | **€36.000** |

---

## Desired Items (D1–D3) — scope condizionale

| Item | Descrizione | Stima NTE |
|------|-------------|-----------|
| D1 | UI modernization | €8.000 |
| D2 | Internationalization (giapponese) | €5.000 |
| D3 | Dark mode | €3.000 |
| **Totale desired** | | **€16.000** |

I desired item vengono attivati su richiesta esplicita di Meridian, uno alla volta, con ordine di lavoro separato.

---

## Assunzioni di pricing

- Fee basate su 8 settimane di ingaggio per i required item (vedi Timeline).
- Include tutto il lavoro di discovery, sviluppo, test e documentazione.
- **Non include (fuori scope, valutabili su richiesta con ordine di lavoro separato):**
  - Infrastruttura e hosting (server, cloud, ambienti di produzione)
  - Licenze software di terze parti (strumenti, servizi esterni, plugin commerciali)
  - Migrazione a database relazionale o documentale — l'architettura attuale è basata su file JSON; se Meridian decide di evolvere verso un DB, lo scope va ridefinito separatamente
- Modifiche di scope in corso d'opera saranno valutate e comunicate per iscritto prima dell'esecuzione.
- Pagamenti: 30% all'avvio, 30% alla milestone settimana 4 (R4+R3 consegnati), 40% alla consegna finale.

---

## Perché fixed-fee

Meridian ha già vissuto un ingaggio che si è concluso con lavoro incompiuto. Il fixed-fee trasferisce il rischio di scope sul vendor — noi. Se un fix richiede più tempo del previsto, il costo è nostro, non vostro. La prevedibilità di budget è una garanzia, non un'aspirazione.
